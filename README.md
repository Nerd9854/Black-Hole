When you open up this project I have setup a simple scene, inisde is:
- Black Hole
- Accrection Disk
- Star in the top left
----
These are used to show you what the renderer is atchually rednering.
Be warned I have set the speed of the sim to be slower to focus on accuracy, so the scene will looks to be doing nothing for a few seconds. BUt it is slowly working in the Backround.
Once the ray has collided with an obejct it will return and write stself to a texture, once the first write attempt ahs happned to the texture the camera will switch to veiwing the texture. It will be clear once the render had finished once the gradiant has disappeared.
